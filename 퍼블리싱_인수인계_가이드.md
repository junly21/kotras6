# 🎨 퍼블리싱 인수인계 가이드

## 📋 프로젝트 개요

**프로젝트명**: KOTRAS6 (한국철도공사 시스템)  
**목적**: 지하철 노선도 시각화 및 정산 데이터 분석 시스템  
**주요 기능**: 노선도 조회, 정산 데이터 분석, 경로 검색, 파일 업로드

---

## 🛠 기술 스택

### Frontend Framework

- **Next.js 15.3.4** (App Router)
- **React 19.0.0**
- **TypeScript 5**

### Styling & UI

- **Tailwind CSS 4.1.10** (CSS Variables 기반)
- **shadcn/ui** (New York 스타일)
- **Radix UI** (기본 컴포넌트)
- **Lucide React** (아이콘)

### 주요 라이브러리

- **AG Grid** (테이블/그리드)
- **Recharts** (차트)
- **OpenLayers (ol)** (지도)
- **svgson** (SVG 파싱)
- **React Hook Form** (폼 관리)
- **Zod** (스키마 검증)
- **Zustand** (상태 관리)

## 🎯 퍼블리싱 작업 범위

### 1. 노선도 관련 페이지 (`/app/network/`)

- **노선도 조회** (`/network/line/`)
  - SVG 기반 인터랙티브 노선도
  - 확대/축소, 드래그, 툴팁 기능
  - 노선별 필터링 (1호선, 2호선, 5호선, 9호선)

### 2. 정산 데이터 페이지 (`/app/settlement/`)

- **정산 개요** (`/settlement/overview/`)
- **기관별 정산** (`/settlement/by-institution/`)
- **역별 정산** (`/settlement/by-station/`)
- **노선별 정산** (`/settlement/by-route/`)
- **OD별 정산** (`/settlement/by-od/`)

### 3. 공통 컴포넌트 스타일링

- FilterForm 컴포넌트 레이아웃 개선
- AG Grid 테이블 스타일링
- Recharts 차트 스타일링
- 버튼, 입력 필드 등 UI 컴포넌트 정리


### 2. UnitRadioGroup 컴포넌트

```tsx
import { UnitRadioGroup } from "@/components/ui/UnitRadioGroup";

// 사용 예시
<UnitRadioGroup
  value={unit}

    { value: "백만", label: "백만원" },

```
