## 🎨 디자인 가이드라인

### 색상 시스템

```css
/* Primary Colors */
--primary: 0 0% 9%; /* #171717 */
--primary-foreground: 0 0% 98%; /* #fafafa */

/* Secondary Colors */
--secondary: 0 0% 96.1%; /* #f5f5f5 */
--secondary-foreground: 0 0% 9%; /* #171717 */

/* Accent Colors */
--accent: 0 0% 96.1%; /* #f5f5f5 */
--accent-foreground: 0 0% 9%; /* #171717 */

/* Muted Colors */
--muted: 0 0% 96.1%; /* #f5f5f5 */
--muted-foreground: 0 0% 45.1%; /* #737373 */

/* Border & Input */
--border: 0 0% 89.8%; /* #e5e5e5 */
--input: 0 0% 89.8%; /* #e5e5e5 */
```

### 타이포그래피

- **제목**: `text-2xl font-bold` (24px, 700)
- **부제목**: `text-xl font-semibold` (20px, 600)
- **본문**: `text-base` (16px, 400)
- **작은 텍스트**: `text-sm` (14px, 400)

### Spacing

- **기본 단위**: 4px
- **컴포넌트 간격**: `gap-4` (16px)
- **섹션 간격**: `gap-6` (24px)
- **페이지 여백**: `p-6` (24px)

### Border Radius

- **기본**: `rounded` (4px)
- **버튼**: `rounded-md` (6px)
- **카드**: `rounded-lg` (8px)

## 📱 반응형 디자인

### Breakpoints

```css
/* Tailwind CSS Breakpoints */
sm: 640px   /* 모바일 가로 */
md: 768px   /* 태블릿 */
lg: 1024px  /* 데스크톱 */
xl: 1280px  /* 큰 데스크톱 */
2xl: 1536px /* 초대형 화면 */
```

### 반응형 레이아웃 가이드

- **모바일 (sm)**: 단일 컬럼, 세로 스택
- **태블릿 (md)**: 2컬럼 그리드
- **데스크톱 (lg+)**: 3-4컬럼 그리드
